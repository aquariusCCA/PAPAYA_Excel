## 🧠 `COUNTIF` 是什麼？

`COUNTIF` 函數會根據你設定的「條件」，計算符合該條件的儲存格個數。  
👉 適合用來做成績統計、分類統計、關鍵字計數等！

---

## 📘 語法

```excel
COUNTIF(range, criteria)
```

- `range`：要檢查的儲存格範圍。
- `criteria`：你要設定的條件（可以是數字、文字、運算式、通配符等）。

---

## ✅ 範例 1：找出成績及格的人數

| A        | B       |
|----------|---------|
| 姓名     | 分數    |
| 小明     | 78      |
| 小華     | 45      |
| 小美     | 90      |
| 小強     | 55      |
| 小美     | 60      |

你可以在其他儲存格輸入公式：

```excel
=COUNTIF(B2:B6, ">=60")
```

### ➤ 結果是 `3`  
因為有 3 位同學分數 ≥ 60（小明、⼩美、⼩美）

---

## ✅ 範例 2：找出某個文字出現幾次

| A        |
|----------|
| 蘋果     |
| 香蕉     |
| 蘋果     |
| 葡萄     |
| 蘋果     |

```excel
=COUNTIF(A1:A5, "蘋果")
```

### ➤ 結果是 `3`  
因為「蘋果」出現了 3 次。

---

## ✅ 範例 3：使用通配符統計

假設你要找出所有以「小」開頭的名字（像是 小明、小美、小強）

| A        |
|----------|
| 小明     |
| 小華     |
| 阿宏     |
| 小美     |
| 小強     |

```excel
=COUNTIF(A1:A5, "小*")
```

### ➤ 結果是 `4`  
`*` 表示「任意多字元」，所以「小*」代表所有「開頭是小」的字串。

---

## ✅ 範例 4：不等於某值

如果你想統計不是「蘋果」的項目數：

```excel
=COUNTIF(A1:A5, "<>蘋果")
```

`<>` 代表「不等於」

---

## 🎯 常用條件對照表

| 條件寫法      | 意思                    |
|---------------|-------------------------|
| `"=蘋果"`     | 等於「蘋果」            |
| `"<>蘋果"`    | 不等於「蘋果」          |
| `">=60"`      | 大於等於 60             |
| `"<100"`      | 小於 100                |
| `"小*"`       | 開頭是「小」的字串       |
| `"*果"`       | 結尾是「果」的字串       |
